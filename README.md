# JupyterLab Theme Inspired by BLΛƆKPIИK

Hello, data enthusiasts who are also _Blinks_! Do you work with JupyterLab every single day while listen to _"How You Like That"_? If so, here I made a simple JupyterLab theme inspired by __BLΛƆKPIИK__. Source code & setup instructions can be found on the link below. Hope you like it & let's collaborate if you have a new idea on this theme, PR on its GitHub repo.

- 📁 Source code 

    - Github repo: [JupyterLab BLΛƆKPIИK Theme](https://github.com/LintangWisesa/JupyterLab_Theme_Blackpink)

    - Preview:

        ![jupyterlab_blackpink](img/jupyterlab_blackpink.png)

<hr>

### 🔧 How to use this theme:
    
- Make sure you've installed __JupyterLab__ (_the new classic Jupyter Notebook interface_)
- Run this command on terminal:
    ```bash
    $ jupyter labextension install @lin26/jupyterlab_blackpink
    ```
- Activate your JupyterLab:
    ```bash
    $ jupyter lab
    ```
- Go to _Settings > JupyterLab Theme_ then choose *JupyterLab Blackpink*. Aye aye~ 😇
